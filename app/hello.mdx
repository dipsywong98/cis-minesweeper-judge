# Minesweeper

Author - [Dipsy](https://github.com/dipsywong98)
 
[Minesweeper](https://www.google.com/search?q=minesweeper&rlz=1C1YTUH_zh-HKHK1048HK1048&oq=minesweeper&gs_lcrp=EgZjaHJvbWUqBwgAEAAYjwIyBwgAEAAYjwIyDwgBEC4YQxiDARixAxiKBTIHCAIQLhiABDIHCAMQABiABDIHCAQQLhiABDIGCAUQRRg8MgYIBhBFGDwyBggHEEUYQdIBCDUxMTlqMGo3qAIAsAIA&sourceid=chrome&ie=UTF-8#cobssid=s) is an interesting game, why not automate it?

## Instructions

You need to expose a POST endpoint `/minesweeper`, that accept a list of url that will lead you to different minesweeper game and you need to respond the array of mine positions in each game in `[rowIndex, columnIndex]` format, sorted from left to right, then top to bottom, where both row and column are 0-indexed.

MIME type of application/json are expected for both request and response.

You may check [/sample](/sample) for more examples input/output. You may download source code of the minesweeper game <a href={MINESWEEPER_UI_DOWNLOAD_URL}>here</a>

## Sample post body

<pre>
  [
    "{MINESWEEPER_UI_URL}#MiwyLHdwQT0=",
    "{MINESWEEPER_UI_URL}#Myw0LHdxUUE="
  ]
</pre>

## Sample response

```json
[
  [[0,0], [1,1]],
  [[0,0], [0,2], [1,1]]
]
```
